IMAGE_CLASSIFICATION_WITH_ARRAYS¶

Dataset Types:

  • tf-keras://

Query Types:

  • (Nx, Ny) vector of integers ranging 0-255, corresponding to pixel values of an query image in grayscale

Prediction Type:

  • Integer, corresponding to the label of the query image
