Using Rafiki’s Admin Web¶

Table of Contents

  • Using Rafiki’s Admin Web

Rafiki Admin Web is accessible at <rafiki_host>:<admin_port> (e.g. 127.0.0.1:3001 by default).

Logging In¶

Log in with the same credentials for Rafiki Admin.
